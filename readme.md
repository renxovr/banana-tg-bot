# Banana Telegram Bot

## Deploy in local
1. You need create **.env**
2. Add:  
    ```
    BOT_TOKEN = [YOUR_TOKEN]
    ```
3. Install **requirements.txt**  
4. Run

## Deploy in Heroku
1. Go to **Settings**
2. Find **Config Vars**
3. Add:
    ```
    KEY = BOT_TOKEN
    VALUE = [YOUR_TOKEN]
    ```
4. Deploy
